## Git Workflow

### Daily Process
1. **Morning:** Pull latest `dev` branch
2. **Branch:** Create `feature/[student]-day[X]` branch
3. **Development:** Work on assigned files (no repetitions)
4. **Commit:** Use provided commit message
5. **Push:** Push to feature branch
6. **PR:** Create Pull Request to `dev`
7. **Merge:** Team lead reviews and merges

### Integration Points
- **End of Day 1-3:** Core foundation integration
- **End of Day 4-6:** Feature integration
- **End of Day 7-9:** Testing and deployment integration
